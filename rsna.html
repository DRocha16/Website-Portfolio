RSNA Bone Age

Why?

The conventional method that a doctor has to assess one's age is to look into his bones (frequently the hand's bones) and use his memory of last diagnostics and of studied cases to give a prediction of the patient's bone age.

So, this method is based on comparation and search for specific features in the x ray images of the patient's hand. Which makes this problem perfectly suited for a computer using Machine Learning and Image Processing methods.

Hence, starting this project was a way of being able to try to solve a problem related to the field of medicine, but also and principally as a way to learn about the field of machine learning by doing.


Graphical User Interface & Image Processing

The goal of this GUI was to be a tool to help in the process of finding the best image processing pipeline for a dataset of X-ray hand's images.

This work, specifically, was designed to segment and isolate the wrist bones, in order to extract meaningful features from there to feed them into a machine learning model later on.

The images used were obtained from the RSNA Bone Age contest. This contest aimed to correctly identify the age of a child from an X-ray of their hand. The whole dataset can be accessed in Kaggle: https://www.kaggle.com/kmader/rsna-bone-age

PyQt5 and Qt Designer were used to create the GUI. All image processing methods were created using Scikit-Image Other packages such as Matplotlib, NumPy, SciPy and Pandas were also used. It was developed as an work for an image processing course.

This GUI was developed as an assignment for an image processing course. Several methods of image processing such as threshold filtering, contrast enhancements region-based segmentation and some created by myself were used.


Lasso Regression & NNs

In the image processing phase, the visual information on the images was transformed into numerical features. Various methods were used such as GLCM features, corner features (Harris Corner Detector), CENSURE features, the percentage of non black pixels and some methods created by myself.

Then, this dataset was used to train a model. A more classical approach was to use Lasso regression, the other approach was using a neural network (made predominantly with ReLu as activation function).

The accuracy of the models were:

- Lasso Regression: MAE of 23.4 months and RMSE of 37.1 months

- Neural Network: MAE of 18.7 months and RMSE of 20.5 months

These models are still work on progress and these values, whenever I learn somethinf new useful to this I try to implement it.
